site-chat

AI-powered support chat for any website. Try the widget in the bottom-right corner.

Try it out

Click the chat bubble and ask a question. The AI will answer based on the knowledge base loaded on the server.

Try asking: "What's your return policy?" or "Do you ship internationally?" or "What payment methods do you accept?"

How it works

This page has a single <script> tag at the bottom. That's all it takes to add the chat widget to any website.

The widget connects to a backend server that forwards messages to an LLM API. The AI uses a knowledge base (plain text files on the server) to answer questions accurately.

Your own setup

Clone the repo, add your FAQ to the data/ folder, configure your API key, and deploy with Docker. Full instructions on GitHub.
